# Election Process Assistant 🗳️

Welcome to our project submission for the **Google Cloud Prompt Wars Buildathon**!

## About Our Project
We have built an **AI-powered Election Process Assistant**, competing in the **Civic Tech & Public Information** vertical. Our goal is to make election processes, voter registration, specific voting requirements, and timelines incredibly accessible and easy to understand for all citizens.

## Key Features
- **Conversational Guidance:** Built with Google **Gemini 1.5 Pro** for sophisticated natural language reasoning and contextual advice.
- **Intent Recognition:** Dynamically categorizes inquiries about registrations, polling locations, or tracking voting steps.
- **Step-by-Step AI Workflows:** Orchestrates detailed checklists dynamically based on the state or context.
- **Offline & Scale Resilience:** Contains an elegant, statefully-separated containerized architecture built natively on Google Cloud Run.
